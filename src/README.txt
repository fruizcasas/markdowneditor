FEATURES / CARACTERISTICAS
--------------------------
- Editor with live preview (updates as you type)
  Editor con vista previa en vivo (se actualiza mientras escribes)
- Right-click in editor to quickly insert elements
  Click derecho en el editor para insertar elementos rapidamente
- Export to PDF / Exportar a PDF
- Copy HTML to paste in Outlook/Word

- Customizable style profiles / Perfiles de estilo personalizables
- Light/dark theme / Tema claro/oscuro

Keyboard shortcuts / Atajos de teclado:
  Ctrl+N          New document / Nuevo documento
  Ctrl+O          Open file / Abrir archivo
  Ctrl+S          Save / Guardar
  Ctrl+Wheel      Zoom editor
  Ctrl+Wheel      Zoom preview (when hovering / sobre preview)
  Right-click     Menu to insert elements / Menu para insertar


REQUIREMENTS / REQUISITOS
-------------------------
- Windows 10/11
- Python 3.9 or higher (auto-detected)
  Python 3.9 o superior (se detecta automaticamente)


EXPORT TO PDF (optional) / EXPORTAR A PDF (opcional)
----------------------------------------------------
To export to PDF you need wkhtmltopdf:
Para exportar a PDF necesitas wkhtmltopdf:

1. Download from / Descarga de: https://wkhtmltopdf.org/downloads.html
2. Copy wkhtmltopdf.exe to the wkhtmltopdf/ folder
   Copia wkhtmltopdf.exe a la carpeta wkhtmltopdf/

  (instructions inside / instrucciones dentro del archivo)
